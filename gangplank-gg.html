Background

This website is going to be a hub for all things related to the League of Legends game character Gangplank. It have a variety of features that I haven't implemented before on any website.

Why?

In high school, I played League of Legends religiously and seriously on the North American server. I peaked in Challenger, a rank on the League of Legends leaderboard reserved for the top 200 players in the region. I only play Gangplank, my favorite character in the game. I made some videos and streamed a bit, contributing a small amount to the community. As I gain this skill of creating websites, I thought I could contribute to something I love in a cool way.

Structure

  • uses the Riot Games api to gather static data as well as store data about Gangplank players in every region. Eventually, I want the database to provide the most in-depth information on one champion than any other website has done before.
  • uses the Reddit api to display recent activity from the Gangplank mains subreddit.
  • I'm also going to attempt to make it a fast loading MVC (Model-View-Controller) based website, which will probably be easiest by use of a JavaScript framework like React or Vue.js, which can handle the state and views of my DOM elements.
Gangplank is a pirate. Yarrr!

This project is meant to increase my skill in working with apis, JavaScript frameworks, and also fulfill a service void in the community by having this website online.

I haven't chosen which javascript framework I want to use, right now I'm leaning towards ReactJS/Express to handle my database queries well. As I learn more about how apis, queries and headers work I will probably have a good idea on how to set this project up better.
